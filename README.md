# Back-end: node Express.js

## Quick start
```
    npm install
    npm start
```

The launched server will be listening on the port 5000.
## System environment variables

### `MONGO_URL`
ex)
```
http://127.0.0.1:27017/therapist
```

### `FONT_URL`
ex)
```
http://127.0.0.1:7000/
```
